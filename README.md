# 🌌 Galaxy Forge

A futuristic wallpaper platform featuring immersive space visuals, interactive animations, and curated gaming and aesthetic wallpapers.

## 🚀 Features

- 🪐 Rotating 3D Planet
- ⭐ Animated Starfield Background
- 🖱️ Mouse Parallax Camera Effects
- 🎮 Gaming Wallpapers
- 🌌 Space Wallpapers
- 🎨 Aesthetic Wallpapers
- 🔍 Live Wallpaper Search
- 📂 Category Filters
- 🎲 Random Wallpaper Generator
- 📱 Fully Responsive Design
- ✨ Modern Glassmorphism UI
- 🚀 GitHub Pages Deployment

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Three.js
- GitHub Pages

## 🌠 Inspiration

Galaxy Forge was inspired by modern space-themed interfaces, gaming launchers, futuristic dashboards, and premium product websites. The goal was to create a visually immersive wallpaper experience while learning advanced front-end development techniques.

## 📸 Highlights

- Interactive 3D environment
- Smooth animations
- Dynamic wallpaper gallery
- Futuristic cosmic aesthetic
- Lightweight and fast-loading architecture
